# AutoClickerPro
A Auto Clicker for games like roblox, get the max clicks with hotkeys to start and stop.
Its a simple Auto Clicker will click where the mouse is located.

# How to use it
1) Download the AutoClickerPro zip file it has a .exe file and can run it just like any other program
   the easiest way to get it running, I used Pyinstaller to create it.
2) You can download or clone the repo and install requirements.txt
   ```python
   pip install -r requirements.txt
3) Build with PyInstaller
   ```python
   pip install -U pyinstaller
4) build in the terminal with PyInstaller
   ```python
   pyinstaller --noconsole --onefile --windowed --icon="lib\icons\pointer_cursor_mouse_touch_click_hand_icon_250747.ico" --name="AutoClickerPro" autoclicker.py
  
# Features
1) Set Custom keys to start and stop AutoClickerPro
2) Hotkeys no need for GUI buttons to start and stop Listens to keypress
